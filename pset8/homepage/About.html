About

Zone is an online social media plattform specially for programmers. It is created with html, css and javascript at first and hopefully add more features on this site. This is a project to be submitted to cs50 course on harvard. It is mainly for the purspose of practicing and familiarizing the structures, how the web work and how websites are created. This site is called ZONE. It is where programmers socialize, create, explore and discover together. The main purpose of this site is to provide programmer more engagement in socializing with others like them, where they can add more and more connections and friends along their journey.
